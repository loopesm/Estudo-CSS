TRABALHANDO COM CSS

Comentarios em CSS são atribuidos entre = /* comentario */

Podemos trabalhar de 3 Formas com CSS que será mostrado abaixo

INLINE - Podemos aplicar os códigos CSS dentro da própria TAG

Para isso, usamos o stributo "style" dentro da TAG

INTERNAL STYLE SHEET -

Para usar a folha de estilo interna do código HTML, temos que colocar a TAG dentro do HEAD

Com a TAG "style" e atributo "type" com valor "=text/css"

Para atribuirmos valor CSS para algo precisamos usar "seletores" que se tratam dos nomes das TAGs que são usadas normalmente

A propriedades serão colocadas dentro de "chaves"

Colocamos então a propriedade "color:" e o valor "red;" com ponto e vírgula para definir que eu terminei a execução"

EXTERNAL STYLE SHEET

Precisamos utilizar um novo arquivo para usar a folha de estialo externa

extenção do arquivo em ".css"

Dentro da TAG "head" devemos usar a TAG "link" conforme abaixo:

< link rel="stylesheet" type="text/css" href="NOME DA LOCALIZAÇÃO DA FOLHA DE ESTILOS" >

UTILIZANDO "CLASSES" E "IDs"

Os "IDs" defines elementos de localização únicas

IDs são inicados com o jogo da velha "#"

Devemos definir dentro das TAGs: id="principal"

Dentro da TAG "style definimos o ID como "#principal" e atribuimos as propriedades


As "CLASSES" servem para definis diversos elementos com uma caracteristica em comum

Classes são iniciadas nos códigos com pontos "."

Devemos definir dentro das TAGs: class="azul"

Dentro da TAG "style" definimos a classe ".azul" e atribuimos as propriedades

TRABALHANDO COM COR E IMAGEM DE FUNDO

• "Background-color" pode determinar uma cor para o plano de fundo de uma propriedade

• Para aplicar uma imagem ao background de ser usado o "background-image: url('...')"

• Para a imagem não se repetir na tela podemos utilizar o comando "background-repeat"

• Para fixar a imagem em relação a rolagemm da página é o "background-attachment: fixed;"

• Para mover a imagem em relação a rolagemm da página é o "background-attachment: scroll;"

• Para alinhamento da imagem deve ser usado o "background-position: center;"

ELEMENTOS BLOCO

Basicamente existe 3 tipos de elementos em bloco:

Elementos "inline", "block", "inline-block"

Para utilizarmos essa função, precisa ser adicionado a propriedade "display" e colocado uma das 3 opções

Elementos BLOCK = Ocupam a tela toda e geralmente se posicionam um abaixo do outro

Elementos INLINE = Ocupam o tamanho do seu conteúdo e ficam um ao lado do outro

Elementos INLINE-BLOCK = Ocupam o tamanho do seu conteúdo porém se posicionam um abaixo do outros

POSICIONAMENTO DE ELEMENTOS

Teremos esses 04 tipos de posicionamento:

STATIC - Posição padrão já aplicada aos elementos

RELATIVE - Para posicionarmos precisamos usar medidas em pixel dentro das propriedades "top", "right", "bottom", "left" para definir onde o elemento vai ficar dentro dos elementos de caixas

ABSOLUTE - Essa posição é calculada com base no conteiner que ele está inserido ( ou com base no body ) ( o container precisa ter uma posição estática ou relativa para poder usar esse atributo )

FIXED - Essa classe é um atributo que deixa o elemento posicionado fixamente com relação a janela do navegador

Para atribuir uma posição precisamos definir a propriedade "position"

FORMATANDO LINKS

Os links tem estados como : "Não visitados" e "Visitados"

Podemos realizar essa diferenciação usando os termos "a:link" ( para não visitados ) e "a:visited" ( para links visitados ) dentro da folha de estilo

Há também o estado chamado "HOVER", que diferencia o link quando passamos o mouse sobre o mesmo, e para isso usamos o "a:hover" dentro do arquivos de estilo

Outro Estado que podemos ter no nosso projeto é o "ATIVO" que é onde muda o aspecto do link ao clicar com o mouse no mesmo, e podemos usar com o "a:active"

Podemos atribuir outros atributos para os links ( backgrounds e etc )

Seguir a sequecia de formatação = link - visited - hover - active
